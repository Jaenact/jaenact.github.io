# Jaenact Portfolio

A modern, "Toss-style" portfolio website for Jaenact, a Security Researcher.

## Features
- **Design**: Clean, minimal, and 3D-oriented design inspired by Toss.
- **Tech Stack**: HTML5, CSS3, Vanilla JavaScript.
- **Animations**: Smooth scroll reveals, magnetic buttons, and 3D tilt effects.
- **Icons**: Professional Phosphor Icons.

## Deployment
This site is designed to be hosted on GitHub Pages.
1. Push to a GitHub repository.
2. Go to Settings > Pages.
3. Select `main` branch as source.
